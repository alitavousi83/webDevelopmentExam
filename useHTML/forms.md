## برگه تقلب آزمون

| نوع ورودی | کاربرد           |
| --------- | ---------------- |
| text      | متن معمولی       |
| password  | رمز عبور         |
| email     | ایمیل            |
| number    | عدد              |
| date      | تاریخ            |
| time      | ساعت             |
| color     | انتخاب رنگ       |
| file      | انتخاب فایل      |
| radio     | انتخاب یک گزینه  |
| checkbox  | انتخاب چند گزینه |
| submit    | ارسال فرم        |
| hidden    | داده مخفی        |

### تگ‌های مهم فرم

| تگ         | کاربرد         |
| ---------- | -------------- |
| `<form>`   | ایجاد فرم      |
| `<input>`  | دریافت ورودی   |
| `<label>`  | برچسب فیلد     |
| `<select>` | لیست کشویی     |
| `<option>` | گزینه‌های لیست |

### سوالات بسیار پرتکرار آزمون

✅ `<form>` برای ایجاد فرم
✅ `action` مقصد ارسال اطلاعات
✅ `method` نوع ارسال (GET / POST)
✅ `radio` انتخاب یک گزینه
✅ `checkbox` انتخاب چند گزینه
✅ `submit` ارسال فرم
✅ `password` برای رمز عبور
✅ `email` برای ایمیل
✅ `file` برای آپلود فایل

این‌ها معمولاً پرتکرارترین سؤالات بخش «فرم‌های HTML و انواع ورودی» هستند.
