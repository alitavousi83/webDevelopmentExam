# سوالات چهارگزینه‌ای استاندارد

## مبحث: عناصر معنایی HTML5

### سوال 1

منظور از عناصر معنایی (Semantic Elements) در HTML5 چیست؟

الف) عناصری که فقط برای زیبایی صفحه استفاده می‌شوند.
ب) عناصری که معنی و نقش محتوای خود را مشخص می‌کنند.
ج) عناصری که فقط در CSS کاربرد دارند.
د) عناصری که فقط برای تصاویر استفاده می‌شوند.

✅ **پاسخ صحیح: ب) عناصری که معنی و نقش محتوای خود را مشخص می‌کنند.**

---

الف) `<div>`

د) `<font>`

✅ **پاسخ صحیح: ج) `<header>`**

## برگه تقلب 30 ثانیه‌ای

| تگ          | کاربرد            |
| ----------- | ----------------- |
| `<header>`  | سربرگ             |
| `<nav>`     | منوی ناوبری       |
| `<main>`    | محتوای اصلی       |
| `<section>` | بخش مرتبط         |
| `<article>` | مقاله / خبر مستقل |
| `<aside>`   | محتوای جانبی      |
| `<footer>`  | پاورقی            |

### سوالات خیلی پرتکرار آزمون

✅ `<header>` = سربرگ
✅ `<footer>` = پاورقی
✅ `<nav>` = منو
✅ `<article>` = محتوای مستقل
✅ `<section>` = بخش مرتبط
✅ `<aside>` = محتوای جانبی
✅ `<main>` = محتوای اصلی
✅ `<div>` = غیرمعنایی (Non-Semantic)
