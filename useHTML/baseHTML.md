# سوالات چهارگزینه‌ای استاندارد

## مبحث: نحوه و ساختار HTML

### سوال 1

HTML مخفف چیست؟

الف) Hyper Text Markup Language
ب) Hyper Transfer Markup Language
ج) High Text Markup Language
د) Hyper Tool Markup Language

✅ **پاسخ صحیح: الف) Hyper Text Markup Language**

---

### سوال 2

وظیفه اصلی HTML چیست؟

الف) برنامه‌نویسی سمت سرور
