### نکات طلایی آزمون

🔹 **Client = درخواست‌کننده (Requester)**
🔹 **Server = ارائه‌دهنده خدمات (Provider)**
🔹 **Request → Server → Response**
🔹 **مرورگر = Client**
🔹 **Apache و Nginx = Web Server**
🔹 **HTTP پروتکل اصلی ارتباط Client و Server**

این نکات معمولاً به‌صورت مستقیم یا مفهومی در آزمون مطرح می‌شوند. بعد از این بخش وارد **HTML** می‌شوی که معمولاً بیشترین تعداد سؤال آزمون از آن طرح می‌شود.
