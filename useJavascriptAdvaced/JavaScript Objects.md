# برگه تقلب 20 ثانیه‌ای

### مفاهیم اصلی

| مفهوم     | معنی                           |
| --------- | ------------------------------ |
| Object    | مجموعه‌ای از داده‌ها و رفتارها |
| Property  | ویژگی یا داده                  |
| Method    | تابع متعلق به شیء              |
| Prototype | سازوکار وراثت در JavaScript    |

---

### نکات طلایی آزمون

✅ Object = مجموعه Property و Method
✅ Property = داده
✅ Method = رفتار یا تابع
✅ Prototype = وراثت (Inheritance)
✅ JavaScript از Prototype-Based Inheritance استفاده می‌کند.
✅ Prototype باعث اشتراک متدها و کاهش تکرار کد می‌شود.
✅ مهم‌ترین کاربرد Prototype = استفاده مجدد از کد (Code Reuse)

🎯 اگر طراح آزمون بخواهد سؤال دام‌دار بدهد، معمولاً روی تفاوت **Property و Method** یا مفهوم **Prototype و وراثت** سؤال مطرح می‌کند.
