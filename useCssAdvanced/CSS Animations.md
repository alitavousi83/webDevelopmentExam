# برگه تقلب 20 ثانیه‌ای

### مفاهیم اصلی

| مفهوم      | کاربرد                   |
| ---------- | ------------------------ |
| Transition | تغییر تدریجی بین دو حالت |
| Animation  | حرکت و تغییر چندمرحله‌ای |
| @keyframes | تعریف مراحل انیمیشن      |

---

### تفاوت مهم

| Transition  | Animation         |
| ----------- | ----------------- |
| دو حالت     | چند مرحله         |
| ساده‌تر     | پیچیده‌تر         |
| مناسب Hover | مناسب حرکات مداوم |

---

### نکات طلایی آزمون

✅ Transition = تغییر تدریجی ویژگی‌ها
✅ Animation = حرکات و تغییرات چندمرحله‌ای
✅ `@keyframes` = تعریف مراحل انیمیشن
✅ Transition بیشتر برای Hover استفاده می‌شود
✅ Animation می‌تواند تکرار شود
✅ هر دو باعث بهبود تجربه کاربری (UX) می‌شوند

این مبحث معمولاً با یک یا دو سؤال مستقیم از **تفاوت Animation و Transition** و یک سؤال از **@keyframes** در آزمون‌ها مطرح می‌شود.
