# برگه تقلب 20 ثانیه‌ای

### مقایسه Flexbox و Grid

| Flexbox              | Grid               |
| -------------------- | ------------------ |
| یک‌بعدی              | دوبعدی             |
| ردیف یا ستون         | ردیف و ستون        |
| مناسب منوها و نوارها | مناسب صفحات پیچیده |
| تراز کردن آسان       | چیدمان شبکه‌ای     |

---

### کاربردها

| نیاز            | ابزار مناسب |
| --------------- | ----------- |
| منوی افقی       | Flexbox     |
| نوار ابزار      | Flexbox     |
| گالری تصاویر    | Grid        |
| طراحی صفحه کامل | Grid        |

---

### نکات طلایی آزمون

✅ Flexbox = چیدمان یک‌بعدی
✅ Grid = چیدمان دوبعدی
✅ Flexbox برای ردیف یا ستون
✅ Grid برای ردیف و ستون همزمان
✅ Flexbox برای منوها و Toolbar
✅ Grid برای صفحات پیچیده و گالری‌ها
✅ هر دو برای طراحی Responsive استفاده می‌شوند

🎯 اگر در آزمون فقط یک سؤال از این مبحث بیاید، معمولاً پاسخ آن یکی از این دو نکته است:

* **Flexbox = یک‌بعدی**
* **Grid = دوبعدی**
