# برگه تقلب 20 ثانیه‌ای

| شبه‌کلاس        | کاربرد                   |
| --------------- | ------------------------ |
| `:hover`        | قرار گرفتن ماوس روی عنصر |
| `:active`       | هنگام کلیک               |
| `:focus`        | دریافت فوکوس             |
| `:visited`      | لینک بازدید شده          |
| `:first-child`  | اولین فرزند              |
| `:last-child`   | آخرین فرزند              |
| `:nth-child(n)` | فرزند شماره n            |

### نکات طلایی آزمون

✅ شبه‌کلاس‌ها با `:` شروع می‌شوند.
✅ `:hover` = حرکت ماوس روی عنصر
✅ `:active` = کلیک روی عنصر
✅ `:focus` = انتخاب فیلد یا عنصر
✅ `:visited` = لینک بازدیدشده
✅ `:first-child` = اولین فرزند
✅ `:last-child` = آخرین فرزند
✅ `:nth-child()` = انتخاب فرزند با شماره مشخص

🎯 از این مبحث معمولاً حداقل یک سؤال مستقیم از **hover / focus / active** و یک سؤال از **first-child / last-child / nth-child** در آزمون می‌آید.
