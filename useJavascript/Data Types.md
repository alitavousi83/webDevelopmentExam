## برگه تقلب 30 ثانیه‌ای

### انواع داده مهم

| نوع داده  | مثال             |
| --------- | ---------------- |
| Number    | 10 ، 25 ، 3.14   |
| String    | "Ali"            |
| Boolean   | true / false     |
| Null      | null             |
| Undefined | متغیر بدون مقدار |

---

### کلمات کلیدی مهم

| مورد       | کاربرد            |
| ---------- | ----------------- |
| `<script>` | افزودن JavaScript |
| var        | تعریف متغیر       |
| true       | درست              |
| false      | نادرست            |
| null       | مقدار خالی        |
| undefined  | بدون مقدار        |

---

### نکات طلایی آزمون

✅ JavaScript = ایجاد پویایی و تعامل
✅ `<script>` = افزودن جاوااسکریپت
✅ Number = عدد
✅ String = متن
✅ Boolean = true / false
✅ Null = مقدار خالی عمدی
✅ Undefined = متغیر مقداردهی نشده
✅ var = تعریف متغیر

این‌ها پرتکرارترین مفاهیم بخش «نحوه جاوااسکریپت و انواع داده» هستند و معمولاً چند سؤال مستقیم از آن‌ها در آزمون مطرح می‌شود.
