# برگه تقلب 20 ثانیه‌ای

### مفاهیم اصلی

| مفهوم     | تعریف                       |
| --------- | --------------------------- |
| Function  | مجموعه‌ای از دستورات        |
| Parameter | متغیر در تعریف تابع         |
| Argument  | مقدار ارسالی هنگام فراخوانی |
| Return    | بازگرداندن نتیجه            |
| Scope     | محدوده دسترسی متغیر         |

---

### انواع Scope

| نوع    | توضیح                    |
| ------ | ------------------------ |
| Global | در کل برنامه قابل دسترسی |
| Local  | فقط در تابع قابل دسترسی  |

---

### نکات طلایی آزمون

✅ `function` = تعریف تابع
✅ Call = اجرای تابع
✅ Parameter = داخل تعریف تابع
✅ Argument = هنگام فراخوانی
✅ `return` = بازگرداندن نتیجه
✅ Scope = محدوده دسترسی متغیر
✅ Global = سراسری
✅ Local = محلی

این ۸ نکته تقریباً هسته اصلی سؤالات بخش «توابع و دامنه» را تشکیل می‌دهند و معمولاً چند سؤال مستقیم از آن‌ها در آزمون مطرح می‌شود.
