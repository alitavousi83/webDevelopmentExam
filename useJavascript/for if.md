# برگه تقلب 20 ثانیه‌ای

### ساختارهای شرطی

| دستور   | کاربرد            |
| ------- | ----------------- |
| if      | بررسی شرط         |
| else    | حالت خلاف شرط     |
| switch  | بررسی چند حالت    |
| case    | هر حالت در switch |
| default | حالت پیش‌فرض      |

---

### حلقه‌ها

| حلقه  | کاربرد                   |
| ----- | ------------------------ |
| for   | تعداد تکرار مشخص         |
| while | تکرار تا برقرار بودن شرط |

---

### نکات طلایی آزمون

✅ `if` = تصمیم‌گیری بر اساس شرط
✅ `else` = در غیر این صورت
✅ `switch` = چند حالت مختلف
✅ `case` = هر حالت در switch
✅ `default` = حالت پیش‌فرض
✅ `for` = تعداد تکرار مشخص
✅ `while` = تکرار تا برقرار بودن شرط
✅ شرط‌ها معمولاً مقدار **true یا false** برمی‌گردانند.

این بخش یکی از محبوب‌ترین بخش‌های طراحان سؤال است و معمولاً چند سؤال مستقیم از تفاوت **if / switch** و **for / while** مطرح می‌شود.
