# برگه تقلب 30 ثانیه‌ای

### عملگرهای حسابی

| عملگر | کاربرد          |
| ----- | --------------- |
| +     | جمع             |
| -     | تفریق           |
| *     | ضرب             |
| /     | تقسیم           |
| %     | باقیمانده تقسیم |

---

### عملگرهای مقایسه‌ای

| عملگر | معنی    |
| ----- | ------- |
| ==    | مساوی   |
| !=    | نامساوی |
| >     | بزرگ‌تر |
| <     | کوچک‌تر |
| =     | انتساب  |

---

### نکات طلایی آزمون

✅ Variable = ذخیره داده
✅ `var` = تعریف متغیر
✅ `=` = انتساب مقدار
✅ `==` = مقایسه برابری
✅ `!=` = نامساوی
✅ `%` = باقیمانده تقسیم
✅ Expression = ترکیب متغیرها، مقادیر و عملگرها

این مبحث معمولاً پایه سؤالات بخش‌های بعدی مثل **ساختارهای کنترلی (if, switch)** و **حلقه‌ها (for, while)** است.
