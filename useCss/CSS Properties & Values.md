## برگه تقلب آزمون

| Property         | کاربرد       |
| ---------------- | ------------ |
| color            | رنگ متن      |
| background-color | رنگ پس‌زمینه |
| font-size        | اندازه فونت  |
| font-family      | نوع فونت     |
| font-weight      | ضخامت متن    |
| text-align       | تراز متن     |
| margin           | فاصله خارجی  |
| padding          | فاصله داخلی  |
| border           | کادر         |
| width            | عرض          |
| height           | ارتفاع       |
| display          | نحوه نمایش   |

### مقادیر مهم

| مقدار  | کاربرد      |
| ------ | ----------- |
| red    | رنگ قرمز    |
| bold   | متن ضخیم    |
| center | وسط‌چین     |
| block  | نمایش بلوکی |
| inline | نمایش خطی   |
| none   | عدم نمایش   |

### سوالات بسیار پرتکرار

✅ color → رنگ متن
✅ background-color → رنگ پس‌زمینه
✅ font-size → اندازه فونت
✅ text-align:center → وسط‌چین
✅ margin → فاصله خارجی
✅ padding → فاصله داخلی
✅ border → کادر
✅ width → عرض
✅ height → ارتفاع
✅ display:none → مخفی کردن عنصر

این موارد معمولاً بیشترین احتمال طرح سؤال در بخش «خواص و مقادیر CSS» را دارند.
