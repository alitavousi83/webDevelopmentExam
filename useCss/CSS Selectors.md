## برگه تقلب 30 ثانیه‌ای

| انتخابگر   | مثال    | کاربرد                      |
| ---------- | ------- | --------------------------- |
| Element    | `p`     | انتخاب تگ‌ها                |
| Class      | `.box`  | انتخاب عناصر دارای class    |
| ID         | `#box`  | انتخاب عنصر دارای id        |
| Universal  | `*`     | انتخاب همه عناصر            |
| Grouping   | `h1,h2` | انتخاب چند عنصر             |
| Descendant | `div p` | انتخاب عناصر داخل عنصر دیگر |

### نکات طلایی

✅ `#` → id
✅ `.` → class
✅ `*` → همه عناصر
✅ `p` → Element Selector
✅ `div p` → Descendant Selector
✅ `h1,h2` → Group Selector
✅ اولویت: **ID > Class > Element**

این نکات تقریباً هسته اصلی سؤالات مبحث «نحوه و انتخابگرهای CSS» هستند و احتمال طرح آن‌ها در آزمون بسیار بالاست.
