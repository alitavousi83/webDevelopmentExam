# برگه تقلب آزمون

| مفهوم             | توضیح                       |
| ----------------- | --------------------------- |
| Responsive Design | نمایش صحیح در همه دستگاه‌ها |
| Media Query       | تغییر استایل بر اساس دستگاه |
| @media            | شروع Media Query            |
| Breakpoint        | نقطه تغییر استایل           |
| min-width         | حداقل عرض                   |
| max-width         | حداکثر عرض                  |

### نکات طلایی

✅ Responsive = سازگار با موبایل، تبلت و دسکتاپ
✅ Media Query = ابزار اصلی طراحی واکنش‌گرا
✅ `@media` = شروع پرس‌وجوی رسانه‌ای
✅ Breakpoint = نقطه تغییر استایل
✅ `min-width` و `max-width` بسیار پرتکرار هستند.

این نکات تقریباً تمام سؤالات رایج بخش «پرس‌وجوهای رسانه‌ای و طراحی واکنش‌گرا» را پوشش می‌دهند. بعد از این بخش وارد **جاوااسکریپت** می‌شوی که معمولاً بخش بزرگی از آزمون را تشکیل می‌دهد.
