# برگه تقلب 20 ثانیه‌ای

### رویدادهای مهم

| رویداد   | کاربرد        |
| -------- | ------------- |
| click    | کلیک          |
| load     | بارگذاری صفحه |
| change   | تغییر مقدار   |
| focus    | ورود فوکوس    |
| blur     | خروج فوکوس    |
| submit   | ارسال فرم     |
| keypress | فشردن کلید    |

---

### انتشار رویداد

| نوع       | مسیر حرکت    |
| --------- | ------------ |
| Capturing | والد ➜ فرزند |
| Bubbling  | فرزند ➜ والد |

---

### نکات طلایی آزمون

✅ Event = اتفاق در صفحه وب
✅ Event Handling = اجرای کد در پاسخ به رویداد
✅ click = کلیک کاربر
✅ load = بارگذاری صفحه
✅ change = تغییر مقدار فیلد
✅ focus = ورود فوکوس
✅ blur = خروج فوکوس
✅ Bubbling = فرزند به والد
✅ Capturing = والد به فرزند

این مبحث معمولاً ۲ تا ۴ سؤال مستقیم در آزمون دارد، مخصوصاً از تفاوت **Capturing** و **Bubbling** و رویدادهای رایج مثل **click، load و change**.
