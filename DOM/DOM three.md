# برگه تقلب 20 ثانیه‌ای

### مفاهیم اصلی DOM

| مفهوم     | معنی                  |
| --------- | --------------------- |
| DOM       | Document Object Model |
| Node      | هر عنصر در درخت       |
| Root      | عنصر html             |
| Parent    | والد                  |
| Child     | فرزند                 |
| Sibling   | عناصر هم‌سطح          |
| Leaf Node | گره بدون فرزند        |

---

### نکات طلایی آزمون

✅ DOM = مدل شی‌گرای سند HTML
✅ ساختار DOM = درختی (Tree)
✅ html = ریشه (Root)
✅ Parent = والد
✅ Child = فرزند
✅ Sibling = هم‌سطح
✅ JavaScript از طریق DOM عناصر صفحه را کنترل می‌کند.

این ۷ نکته تقریباً تمام سؤالات پایه‌ای بخش **DOM Tree** را پوشش می‌دهند و معمولاً مستقیماً در آزمون مطرح می‌شوند.
