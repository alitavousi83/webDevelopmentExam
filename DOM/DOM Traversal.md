### پیمایش DOM (Traversal)

| مفهوم     | معنی               |
| --------- | ------------------ |
| Parent    | والد               |
| Child     | فرزند              |
| Sibling   | هم‌سطح             |
| Traversal | حرکت بین عناصر DOM |

---

### اصلاح DOM (Modification)

| عملیات  | مثال               |
| ------- | ------------------ |
| Add     | افزودن عنصر        |
| Remove  | حذف عنصر           |
| Update  | تغییر متن یا محتوا |
| Replace | جایگزینی عنصر      |

---

### نکات طلایی آزمون

✅ Traversal = پیمایش بین عناصر DOM
✅ Modification = تغییر عناصر DOM
✅ Parent = والد
✅ Child = فرزند
✅ Sibling = هم‌سطح
✅ افزودن عنصر = Modification
✅ حذف عنصر = Modification
✅ تغییر متن عنصر = Modification
