# برگه تقلب 30 ثانیه‌ای

### متدهای انتخاب عنصر

| متد                      | کاربرد           |
| ------------------------ | ---------------- |
| getElementById()         | انتخاب با id     |
| getElementsByClassName() | انتخاب با class  |
| getElementsByTagName()   | انتخاب با تگ     |
| querySelector()          | اولین عنصر منطبق |
| querySelectorAll()       | همه عناصر منطبق  |

---

### دستکاری DOM

| ویژگی     | کاربرد            |
| --------- | ----------------- |
| innerText | تغییر متن         |
| innerHTML | تغییر محتوای HTML |
| style     | تغییر استایل      |

---

### نکات طلایی آزمون

✅ `getElementById()` → انتخاب با id
✅ `getElementsByClassName()` → انتخاب با class
✅ `getElementsByTagName()` → انتخاب با تگ
✅ `querySelector()` → اولین عنصر منطبق
✅ `querySelectorAll()` → همه عناصر منطبق
✅ `innerText` → متن
✅ `innerHTML` → HTML
✅ `style` → تغییر ظاهر

این موارد تقریباً هسته اصلی سؤالات بخش **انتخاب و دستکاری DOM** را تشکیل می‌دهند و احتمال طرح سؤال از آن‌ها بسیار زیاد است.
